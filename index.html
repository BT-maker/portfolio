MERN stack

(MongoDB, Express.js, React, Node.js) kullanılarak geliştirilen bu haber portalı, modern web teknolojileriyle güçlü bir full-stack uygulama örneği sunuyor. Frontend'de performans odaklı Vite + React yapısı ve responsive tasarım için Tailwind CSS kullanılırken, sayfa yönlendirmeleri React Router ile yönetiliyor. Kullanıcı güvenliği, JWT tabanlı kimlik doğrulama ve HTTP-only cookie'lerle sağlanıyor. Haberler kategorilere göre filtrelenebiliyor ve ana sayfada dinamik reklam alanları yer alıyor. Admin paneli, yalnızca yetkili kullanıcıların erişimine açık olup CRUD işlemleriyle haber, kategori ve reklam yönetimine imkan tanıyor. Veri yönetimi için React Query, backend işlemleri için Express.js ile oluşturulan RESTful API ve NoSQL veritabanı olarak MongoDB kullanılıyor. Ölçeklenebilir ve güvenli yapısıyla profesyonel bir haber platformu deneyimi sunan bu proje, full-stack geliştirme süreçlerini kapsamlı şekilde yansıtıyor.

The Higher Lower Game

Bu proje, kullanıcıların sosyal medya hesaplarının takipçi sayılarını tahmin etmesine dayanan interaktif bir web tabanlı oyundur. Tamamen HTML, CSS ve JavaScript kullanılarak geliştirilen oyun, kullanıcıların karşılarına çıkan iki farklı sosyal medya hesabından hangisinin daha fazla takipçisi olduğunu tahmin etmesi prensibiyle çalışır. Her doğru tahmin oyuncunun skorunu artırırken, yanlış tahmin oyunun sonlanmasına neden olur. Modern ve minimalist bir Türkçe arayüze sahip olan oyun, tüm cihazlara uyumlu responsive tasarımıyla dikkat çeker. Oyun deneyimini zenginleştirmek için SweetAlert2 kütüphanesi kullanılarak görsel açıdan etkileyici bildirimler ve geri bildirimler eklenmiştir. Basit ama bağımlılık yapıcı oyun mekaniği sayesinde kullanıcılar hem keyifli vakit geçirebilir hem de sosyal medya istatistikleri hakkında fikir sahibi olabilirler. Temel web teknolojilerinin etkili bir şekilde kullanıldığı bu proje, interaktif ve eğlenceli bir kullanıcı deneyimi sunmayı hedeflemektedir.
